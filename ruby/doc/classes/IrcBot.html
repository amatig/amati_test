main_loop()

Fa partire il ciclo principale del bot che legge i messaggi provenienti dal socket e li manda ad un nuovo thread chiamando il metodo dispatch.

Nel caso si tratta di un PING del server risponde subito col PONG per non far cadere la connessione.

[Source]

    # File lib/ircbot.rb, line 83
83:   def main_loop()
84:     while true
85:       return if @irc.eof
86:       msg = @irc.gets
87:       Thread.new do
88:         msg = msg.strip
89:         if msg =~ /^PING :(.+)$/i
90:           puts "[ Server ping ]"
91:           send "PONG :#{$1}"
92:         else
93:           dispatch msg
94:         end
95:       end
96:     end
97:   end
