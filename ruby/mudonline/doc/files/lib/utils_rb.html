a_d(type, text)

Ritorna una stringa rappresentante l‘articolo determinativo della parola nell‘argomento text, l‘argomento type invece e’ un intero che rapprensenta il tipo di parola:

  1. Maschile singolare
  2. Maschile plurale
  3. Femminile singolare
  4. Femminile plurale

[Source]

     # File lib/utils.rb, line 99
 99: def a_d(type, text)
100:   con = ??{ b c d f g h j k l m n p q r s t v w x y z }
101:   voc = ??{ a e i o u }  
102:   # 1 maschile singolare e 2 plurale
103:   # 3 femminile singolare e 4 plurale
104:   case Integer(type)
105:   when 1
106:     if text =~ /^(z|pn|gn|ps|x|y)/i
107:       return "lo "
108:     elsif text =~ /^s(.)/i and (con.include? $1.downcase)
109:       return "lo "
110:     elsif (voc.include? text[0,1].downcase)
111:       return (text !~ /^(io|ie)/i) ? "l'" : "lo "
112:     elsif text =~ /^h(.)/i and (voc.include? $1.downcase)
113:       return "l'"
114:     else
115:       return "il "
116:     end
117:   when 2
118:     return "gli " if text.downcase == "dei" # eccezione
119:     if text =~ /^(z|pn|gn|ps|x|y)/i
120:       return "gli "
121:     elsif text =~ /^s(.)/i and (con.include? $1.downcase)
122:       return "gli "
123:     elsif (voc.include? text[0,1].downcase)
124:       return "gli "
125:     elsif text =~ /^h(.)/i and (voc.include? $1.downcase)
126:       return "gli "
127:     else
128:       return "i "
129:     end
130:   when 3
131:     if (voc.include? text[0,1].downcase) and text !~ /^(io|ie)/i
132:       return "l'"
133:     elsif text =~ /^h(.)/i and (voc.include? $1.downcase)
134:       return "l'"
135:     else
136:       return "la "
137:     end
138:   when 4
139:     return (text[0,1].downcase == "e") ? "l'" : "le "
140:   end
141: end

color(c, text)

Ritorna una stringa con i codici colore (per il client irc). L‘argomento c e’ un symbol/stringa per ottenere il colore dall‘insieme indicizzato dei colori disponibili.

Valori possibili di c:

  • black
  • navy_blue
  • green
  • red
  • brown
  • purple
  • olive
  • yellow
  • lime_green
  • teal
  • aqua_light
  • royal_blue
  • hot_pink
  • dark_gray
  • light_gray
  • white

list(array)

Ritorna una stringa, concatenando tramite virgole gli elemeneti di un array. L‘ultimo elemento viene concatenato per ‘e’.

es.
list ( [ "mario", "carlo", "fabio" ] ) #=> "mario, carlo e fabio"

[Source]

    # File lib/utils.rb, line 20
20: def list(array)
21:   str = array.join(", ")
22:   i = str.rindex(", ")
23:   str[i, 2] = " e " if i
24:   return str
25: end
